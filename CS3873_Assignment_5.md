## Question 1 - Address Spaces

The size of each address space is:

- MAC $\rightarrow$ 6 bytes $=$ 48 bits meaning $2^{48} \approx 2.8 \times 10^{14}$ addresses
- IPV4 $\rightarrow$ 4 bytes $=$ 32 bits
- IPV6 $\rightarrow$ 16 bytes $=$ 128 bits
